🔧 技术实现

在Gradio中，简化播放器通过以下配置实现：

audio = gr.Audio(
    label="", 
    value=None, 
    interactive=False, 
    scale=3, 
    show_label=False,
    show_download_button=False,        # 隐藏下载按钮
    show_share_button=False,           # 隐藏分享按钮
    waveform_options={                 # 波形选项
        "show_controls": False, 
        "show_recording_waveform": False
    },
    container=False,                   # 移除容器边框
    elem_classes=["simple-audio-player"] # 自定义CSS类
)

配合CSS样式：

.simple-audio-player audio {
    height: 32px !important;           # 设置紧凑高度
    border-radius: 6px !important;     # 圆角边框
}
.simple-audio-player .audio-container {
    padding: 4px !important;           # 减少内边距
}
.simple-audio-player .audio-waveform {
    display: none !important;          # 隐藏波形显示
}
